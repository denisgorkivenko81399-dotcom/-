# Живая история: малые музеи Ставрополья

## Развертывание

1. Установите Python 3.9+.
2. Склонируйте репозиторий.
3. Установите зависимости: `pip install -r requirements.txt`
4. Запустите: `python app.py`
5. Откройте в браузере http://localhost:5000

## Для публичного доступа (Render)

- Создайте новый Web Service на Render, подключите GitHub репозиторий.
- Build Command: `pip install -r requirements.txt`
- Start Command: `python app.py`
- Сервер будет доступен по HTTPS.

## Интеграция с МАХ

- В кабинете разработчика МАХ создайте мини-приложение.
- Укажите URL вашего сервера (например, https://ваш-проект.onrender.com).
- Приложение будет открываться в WebView.

## Функции

- Карта музеев (OpenStreetMap)
- Список экспонатов с викторинами
- Анонсы событий
- Подписка на новости музеев
- Запись на экскурсию
- Паспорт краеведа (награда за прохождение всех викторин)
- Админ-панель: добавление/редактирование музеев, экспонатов, событий, вопросов
